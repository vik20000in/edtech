Integers - Questions and Answers

Q1: What are integers?
Ans: Integers are a collection of numbers which include whole numbers and negative numbers (e.g., -3, -2, -1, 0, 1, 2, 3).
Q2: What is the additive inverse of -10?
Ans: The additive inverse of -10 is 10.
Q3: What is the absolute value of -5?
Ans: The absolute value of -5 is 5 (written as |-5| = 5).
Q4: Solve: (-15) + (-7)
Ans: -22.
Q5: Solve: (-20) - (-10)
Ans: -20 + 10 = -10.
Q6: What is the product of (-3) x (-4)?
Ans: 12. (Negative x Negative = Positive).
Q7: What is the product of (-5) x 6?
Ans: -30. (Negative x Positive = Negative).

Ans: -5.
Q9: Divide: (-20) รท (-5)
Ans: 4.
Q10: Which property is shown by: a + b = b + a?
Ans: Commutative Property of Addition.
Q11: Which property is shown by: a x (b + c) = (a x b) + (a x c)?
Ans: Distributive Property of Multiplication over Addition.
Q12: What is the additive identity for integers?
Ans: 0 is the additive identity (a + 0 = a).
Q13: What is the multiplicative identity for integers?
Ans: 1 is the multiplicative identity (a x 1 = a).
Q14: Arrange in ascending order: -5, 2, -8, 0, 4
Ans: -8, -5, 0, 2, 4.
Q15: The sum of two integers is -20. If one of them is 10, find the other.
Ans: Let the other integer be x. x + 10 = -20 => x = -20 - 10 = -30.
